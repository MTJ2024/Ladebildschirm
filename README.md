# Greenzone420 - Die Blüte des Feuers

FiveM Loading Screen for Greenzone420 Server

## ⚠️ FiveM Installation - WICHTIG!

**LESEN:** [FIVEM_INSTALL.md](FIVEM_INSTALL.md)

**Kurz:** Kopiere NUR `fxmanifest.lua` + `dist/` zum FiveM Server!

**NICHT** den ganzen Ordner kopieren - sonst versucht FiveM zu builden!

---

## Local Development

```bash
npm install
npm run dev      # Development server
npm run build    # Build for FiveM
```

## Features

- Modern, bold design with large visual elements
- Fire-themed ritual circle animation
- Progressive loading stages (5 stages)
- FiveM compatible loadscreen
- Pre-built for easy deployment
